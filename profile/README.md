## 🤝 Contributing

We welcome contributions from developers at all experience levels. Here's how to get started:

1. **Explore** — Browse our [repositories](https://github.com/korvexia-org) and find one that interests you.
2. **Fork** — Fork the repository you want to contribute to.
3. **Branch** — Create a feature branch: `git checkout -b feature/your-feature-name`
4. **Code** — Make your changes following the project's coding style.
5. **Test** — Ensure your changes are tested and existing tests pass.
6. **Pull Request** — Open a PR with a clear description of your changes.

We review PRs promptly and provide constructive feedback. Beginner-friendly issues are labeled with `good first issue` — a great place to start!
